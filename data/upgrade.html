Hello! Image Resizer/Scaler has successfully been upgraded to version 1.7

1.7 Upgrade

Version 1.7 introduces the following changes:

  • Fixed: Option "Display Image Tooltip" now works again on newer Firefox builds.
  • New: All option changes except for "Display Image Tooltip" will now take affect immediately.

1.6 Upgrade

Version 1.6 introduces the following changes:

  • Fix: Fixed an issue with text editors on certain forums where styles were inserted into the post's content.
  • Fix: Fixed an issue where Image Resizer is not active when it should.
  • New: Image Resizer will now work on existing pages upon installation.

1.5 Upgrade

Version 1.5 introduces the following changes:

  • New: An option to "End resizing if the mouse leaves an image's boundary", false on default.
  • New: An option to "Prevent images from being resized below 32 pixels in size", false on default.

1.4 Upgrade

Version 1.4 introduces the following changes:

  • New: You can now resize images using your mouse's wheel. Hold your configured mouse button while doing so.
  • New: Images that are still loading such as a .gif can now be resized.

1.3.1 Upgrade

Version 1.3.1 introduces the following changes:

  • Updated the add-on to use the latest Add-on SDK built.

1.3 Upgrade

Version 1.3 introduces the following changes:

  • New: An option to swap the dragging button from RMB back to LMB
  • New: An option to disable Image Resizer's tooltips
  • New: An option to enable free resizing mode(or hold down ALT while resizing)
  • New: An option to switch Ctrl(to disable) to Ctrl(to enable)

You can find those options at the add-ons page, which you can access by doing Ctrl+Shift+A

1.2 Upgrade

Version 1.2 introduces the following changes:

  • Changed: Swapped the resizing button from LeftMouseButton to the RightMouseButton.
  • Changed: If an image has been resized, you can right click it to restore the image to its original size, if instead you drag then you'll continue scaling the image.
  • New: Images now have a tooltip to indicate whether the image can be resized or restored to its original size.
  • New: While resizing an image, a dashed outline will now be shown etc!
  • Improved: Changes are no longer applied to the image but to a specified custom class.
  • Improved: Images smaller than 32 pixels wide or length can no longer be resized.
  • Improved: Images loaded post page-loading can now be resized as well!
  • Removed: Double Click no longer maximizes images, use CTRL+MiddleMouseButton instead(Not recommended).

NOTE: To rescale an image, you will have to drag with the right mouse button instead of the left mouse button!
